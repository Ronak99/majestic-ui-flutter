## Installation

### Add majestic_ui

```bash
dart pub global activate majestic_ui
```

### Add your first widget

```bash
majestic_ui add star_rush_background
```

Checkout the [documentation](https://majesticui.com/widgets/star_rush_background) for Star Rush Background.
